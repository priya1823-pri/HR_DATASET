**HR Attrition Analysis Dashboard (Power BI):**

Tools: Power BI, DAX, Excel, Data Modeling, ETL
Role: Data Analyst / BI Analyst

**Project Overview:**
Developed an interactive HR Attrition Analytics dashboard to help HR leadership monitor workforce trends, analyze salary distributions, and identify factors contributing to employee turnover. The solution provided a data-driven foundation for optimizing retention strategies and improving workforce planning.

**Key Contributions:**

Designed a comprehensive Power BI dashboard featuring KPIs such as highest/lowest salary, average tenure, hike percentage, and seniority-level distribution.

Built advanced DAX measures to calculate attrition rates, salary summaries by gender, and employee counts across experience bands and counties.

Created interactive visualizations including time-series attrition trends, demographic breakdowns, geographic employee distribution (map), and salary analysis by experience and county.

Implemented slicers for age groups, experience ranges, salary bands, and counties to enable multi-level drill-down analysis.
